# BrowserStudio v1.7.19

## 主要更新

- 扩展管理的手动分配增加全选、取消全选，以及按环境编号、名称、环境 ID 搜索。
- 支持用逗号输入多个环境编号，并一键选择或取消当前搜索结果。
- 修复同步工具在主控打开网页二级窗口或扩展弹窗后，其他环境操作同步丢失的问题。
- 二级弹窗现在会按照页面 URL、弹窗路径和标题匹配到各环境对应页面，不再错误操作原标签页。
- 优化主控与跟随窗口尺寸不一致时的弹窗鼠标、键盘、拖动和滚轮坐标映射。
- 增加 `Ctrl +`、`Ctrl -`、`Ctrl 0`、数字键盘加减及 `Ctrl + 滚轮`的页面缩放同步。
- 修复 NSIS 卸载区调用普通安装函数导致私有完整安装器编译失败的问题。

## 升级说明

- 已安装 v1.7.x 的用户可直接在客户端检查更新并升级，不需要先安装 v1.7.18。
- 在线升级只替换 BrowserStudio 主程序，现有环境、浏览器内核、扩展、代理池和本地数据保持不变。
- 如果 v1.7.18 客户端或旧更新器已经无法正常工作，请下载 `BrowserStudio-Repair-Upgrade-v1.7.19.ps1` 执行修复升级；它只替换主程序和更新器，升级前会备份旧程序、配置及数据库关键文件，失败时自动回滚。
- 修复升级前请保存工作并关闭所有 BrowserStudio 管理的浏览器窗口；不要先卸载旧客户端。
- 全新安装或缺少浏览器内核的电脑应使用由可信 Windows 打包电脑生成的私有完整安装器。

## 兼容性

- Windows 10/11 x64。
- 源码构建要求 Go 1.25 或更高版本、Node.js 20 或更高版本。
