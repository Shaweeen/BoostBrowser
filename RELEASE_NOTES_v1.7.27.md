# BrowserStudio v1.7.27

## 备份 data 全环境覆盖恢复

- 新客户端的 data 未保留或用户已预先创建空环境时，允许备份 data 覆盖现有环境。
- 覆盖目标按数据文件夹名、旧 Profile ID、环境编号的优先级依次匹配；没有匹配目标的备份环境仍可新建恢复。
- 覆盖时恢复旧环境名称与编号、指纹参数、代理设置、标签、关键字、窗口位置和最后页签。
- 浏览器数据目录整体替换，因此账号、Cookies、Local Storage、IndexedDB、扩展本地存储与钱包扩展本地数据一起恢复。
- 保留当前客户端的目标 Profile ID、实际数据目录、可用内核与当前分组，避免旧内核路径或旧 ID 导致新客户端无法启动。

## 安全与回滚

- 覆盖前仍要求关闭所有环境并进行用户确认。
- 现有数据目录会先移入 `data/recovery-backups/.../overwritten-data`；文件复制或数据库写入失败时自动回滚。
- 只恢复数据和环境配置，不会用备份覆盖当前程序文件、更新器或内核程序。
