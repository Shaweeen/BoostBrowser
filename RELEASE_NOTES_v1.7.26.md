# BrowserStudio v1.7.26

## 旧 data 识别修复

- 修复备份中存在大量环境数据，但因环境目录位于 `profiles\UUID`、`browser-data\UUID` 等嵌套层级而无法识别的问题。
- 文件夹扫描由仅检查 data 第一层，升级为最多四层的有界扫描，最多检查 20,000 个目录、识别 5,000 个环境。
- 使用 Chrome `Local State`、`Default\Preferences`、History 和 Cookies 等结构交叉判断，避免把缓存或普通文件夹误识别为环境。
- 当备份内的旧 `app.db` 损坏、结构不兼容或没有环境记录时，自动降级为文件夹结构识别，不再直接失败。
- `app.db` 清单可用时，仍会扫描并补充数据库中漏掉的实际环境文件夹。
- 修复前端只显示“旧 data 识别失败”的问题，现在会显示后端返回的具体路径、数据库或扫描错误。

## 保护限制

- 扫描不会读取或输出 Cookies 内容，只检查特征文件是否存在。
- 符号链接目录、Chrome 运行锁文件与缓存目录仍会跳过。
- 同名环境覆盖仍需用户确认，并在覆盖前创建回滚备份。
