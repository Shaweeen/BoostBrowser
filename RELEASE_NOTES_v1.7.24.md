# BrowserStudio v1.7.24

## 本次修复

- 完整删除旧的第三方环境导入功能：前端按钮、文件选择框、运行进程扫描、缓存目录探测、配置字段、后端接口、日志与错误文案均不再存在。
- 修复公共 Manager 安装包会删除已有 `chrome\google-148.0.7778.167` 的问题。
- Manager 安装和在线更新现在始终保留用户已有浏览器内核；不携带 Google 148 的安装包也不会再删除该目录。
- 私有完整安装包只有在自身已包含通过兼容标记校验的 Google Chrome 148 内核时才会替换旧目录，避免出现“删除后没有文件补回”。

## Google Chrome 148 恢复

如果旧安装器已经删除了 Google 148 目录，需要安装一次 v1.7.24 私有完整安装包来恢复内核文件。仅使用轻量在线更新不会凭空恢复已经丢失的数百 MB 内核目录。

恢复后在“内核管理”中点击“扫描内核”，应显示：

- Chromium 146
- Google Chrome 148.0.7778.167（内置隔离）

## 升级兼容性

- Windows 10/11 x64。
- v1.7.x 可直接升级到 v1.7.24，无需逐级安装。
- 更新保留现有环境、账号、Cookies、代理池、扩展与用户数据。
