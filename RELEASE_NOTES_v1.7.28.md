# BrowserStudio v1.7.28

## 安装与 data 兼容

- 完整安装器和在线更新只替换程序文件，不覆盖现有 `config.yaml`、`data`、浏览器内核或扩展目录。
- 全新安装未发现 `data` 时自动创建当前版本的空数据结构；已有 `data` 时直接使用，并由最新数据库迁移组件升级读取结构。
- 误卸载默认保留安装目录中的配置、`data`、内核和扩展文件，重新安装到同一路径后可继续识别。
- 当旧数据库清单缺失、但 `data` 中仍有完整 Chrome 环境目录时，启动会原地识别并挂接环境，不复制或改写 Cookies、Local Storage、IndexedDB 与钱包扩展数据，并显示识别结果。

## 钱包扩展更新兼容

- Rabby、MetaMask、Jupiter 等钱包扩展使用相同官方扩展 ID 更新时，只原子替换扩展程序包；各环境内的钱包存储保持不变。
- 扩展更新记录旧版与新版 manifest 版本，并保留一份上一版程序包用于故障回滚。
- 扩展 ID 变化视为另一个扩展，不能自动读取旧 ID 的隔离存储；跨 Windows 账号时，受 DPAPI 加密保护的 Cookie 或密码仍可能要求重新登录。

## 数据安全

- 旧数据识别以当前客户端程序、启动文件和数据库结构为准，不允许旧备份覆盖新版程序组件。
- 自动识别只在当前环境清单为空时执行，避免重复创建或覆盖现有环境。
- 新增安装器数据保护、扩展版本回滚和原地 data 识别测试。
